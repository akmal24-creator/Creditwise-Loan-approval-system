Project Overview

Developed a machine learning-based loan approval prediction system, CreditWise, designed to assist financial institutions in evaluating loan applications based on applicants' financial and demographic information. The system analyzes factors such as credit score, income, debt-to-income ratio, existing loans, collateral value, employment status, and loan purpose to predict whether a loan application is likely to be approved.

Key Contributions

Performed data preprocessing, including missing value handling, feature engineering, and categorical variable encoding using One-Hot Encoding.
Conducted exploratory data analysis (EDA) and correlation analysis to identify key factors influencing loan approval decisions.
Built and evaluated multiple classification models, including Logistic Regression, Decision Trees, and Random Forest Classifiers.
Applied feature scaling and model evaluation techniques using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix.
Identified Credit Score and Debt-to-Income (DTI) Ratio as the most influential features affecting loan approval outcomes.
Compared model performance and selected the best-performing classifier for loan approval prediction.
